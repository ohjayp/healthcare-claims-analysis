# Healthcare Claims Analysis
**CMS Medicare Provider Utilization & Payment Analysis — California 2023**

## Overview
Analyzed 809,374 California Medicare provider records using SQL Server, with interactive dashboards published to Tableau Public.

## Live Dashboards
- [California Medicare Overview](https://public.tableau.com/app/profile/john.paul.garcia/viz/CaliforniaHealthcareInfo2023/Overview)
- [California Medicare Deep Dive](https://public.tableau.com/app/profile/john.paul.garcia/viz/CaliforniaHealthcareInfo2023/DeepDive)

## Tools
SQL Server · Tableau Public · Python · Excel

## Key Findings
- Ambulatory Surgical Centers average $1,286 per service — highest in California
- Los Angeles accounts for $686M in estimated Medicare spend
- Clinical Laboratory is the #1 specialty by total spend at $159M
- Urology has the widest payment range ($42,059) driven by a single outlier provider in La Mesa
